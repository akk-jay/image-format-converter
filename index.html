关于这款图片格式转换工具

这是一款完全免费的在线图片格式转换工具，所有转换操作均在您的浏览器本地完成，不会上传任何图片到服务器。无论您需要将 PNG 转为 JPG、WebP 转为 PNG，还是将苹果 HEIC 照片转为 JPG，都可以在这里快速完成。

为什么选择我们？

  • 隐私安全：图片仅在本地浏览器处理，不上传、不存储
  • 完全免费：无次数限制、无水印、无需注册
  • 格式全面：支持 JPG、PNG、WebP、AVIF、ICO、HEIC
  • 批量处理：支持多张图片一次转换，打包下载
  • 断网可用：首次加载后可离线使用

常见问题

Q：转换图片真的不会上传到服务器吗？
A：是的。本工具使用浏览器 Canvas API 在本地完成所有转换，您的图片永远不会离开您的设备。您可以断开网络后继续使用，以验证这一点。

Q：支持哪些图片格式？
A：支持 JPG、PNG、WebP、AVIF、ICO 格式输出，以及 HEIC/HEIF 格式输入。其中 AVIF 需要现代浏览器支持（Chrome 85+、Firefox 93+）。

Q：可以批量转换吗？
A：可以。展开"批量转换"卡片，上传多张图片即可一键统一转换格式，支持单独下载或 ZIP 打包下载。
