隐私政策

生效日期：2026 年 7 月 8 日

1. 数据处理方式

本工具所有图片转换操作均在您的浏览器本地完成，使用 Canvas API 处理。我们不会将您的图片上传至任何服务器，也不会存储、查看或分享您的任何图片数据。

2. Cookie 使用

本网站使用 Cookie 仅用于以下目的：

  • 流量统计：了解页面访问量、来源等匿名数据
  • 广告展示：Google AdSense 等广告平台可能需要设置 Cookie 以展示相关广告

Cookie 不会用于追踪个人身份或收集个人信息。

3. 第三方服务

本网站可能使用以下第三方服务：

  • Google AdSense：展示广告，Google 可能使用 Cookie 进行广告个性化。Google 广告政策
  • Cloudflare Web Analytics：匿名流量统计，不设置 Cookie。Cloudflare 隐私政策
  • CDN 资源：JSZip、Iconify、heic2any 通过 jsDelivr 和 cdnjs 加载，这些 CDN 服务可能有各自的隐私政策

4. 您的权利

根据 GDPR 及其他隐私法规，您有权：

  • 拒绝非必要 Cookie
  • 请求删除您的数据（本网站不存储任何用户数据，无需删除）
  • 通过浏览器设置管理或清除 Cookie

5. 联系方式

如有隐私相关问题，请通过 GitHub Issues 联系我们：github.com/akk-jay/image-format-converter/issues
