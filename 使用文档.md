# 图片格式在线转换工具 — 使用文档

## 简介

一款**完全免费、无需注册**的在线图片格式转换工具。所有转换在浏览器本地完成，**不上传服务器、不存储任何数据**。支持 JPG、PNG、WebP、AVIF、ICO 格式互转，以及 HEIC 解码、批量处理和 Base64 互转。

---

## 快速开始

1. 打开网页，直接**拖拽图片**到上传区域，或点击上传区选择文件
2. 点击目标格式标签（如 **JPG**、**WebP**）
3. 拖动质量滑块调节输出质量（1%–100%）
4. 点击 **「转换并下载」** 按钮
5. 浏览器自动下载转换后的图片

---

## 功能详解

### 一、单张图片转换

**适用场景**：日常格式切换，如 PNG 转 JPG、WebP 转 PNG 等。

| 步骤 | 操作 |
|------|------|
| 上传 | 拖拽或点击上传区域 |
| 选格式 | 点击顶部格式标签：`PNG` `JPG` `WebP` `AVIF` `ICO` |
| 调质量 | 拖动滑块，数值越高画质越好、文件越大 |
| 转换 | 点击 **「转换并下载」** |

**格式说明**：

| 格式 | 特点 | 推荐场景 |
|------|------|----------|
| **PNG** | 无损压缩，支持透明背景 | 图标、Logo、需透明底的图片 |
| **JPG** | 有损压缩，体积小 | 照片、网页展示、通用场景 |
| **WebP** | 兼顾画质与体积 | 网站图片优化，替代 JPG/PNG |
| **AVIF** | 新一代格式，压缩率极高 | Chrome 85+/Firefox 93+ 的现代网页 |
| **ICO** | 图标格式，含多尺寸 | 网站 Favicon、应用图标 |

> ⚠️ 单张图片上限 **10MB**，超过会提示压缩后再上传。

---

### 二、批量转换

展开 **「批量转换」** 卡片使用。适用于多张图片统一转为一个格式。

1. **拖拽或选择多张图片**（最多 10 张）
2. 选择统一的目标格式
3. 点击 **「批量转换」**
4. 单张结果自动下载，或多张点击 **「打包下载 (ZIP)」**

> 💡 转换结果列表展示每张图的原始大小和转换后大小，方便对比压缩效果。

---

### 三、Base64 互转

展开 **「Base64 互转」** 卡片使用。适用于开发者场景。

**图片 → Base64**：
1. 选择「图片 → Base64」标签
2. 上传图片
3. Base64 编码自动生成
4. 点击 **「复制 Base64」** 一键复制到剪贴板

**Base64 → 图片**：
1. 选择「Base64 → 图片」标签
2. 粘贴 Base64 字符串（支持 `data:image/...;base64,` 前缀）
3. 点击 **「解码并下载图片」**
4. 自动识别格式并下载

---

### 四、HEIC 专项转换

展开 **「HEIC 专项转换」** 卡片使用。适用于苹果设备照片转换。

1. 上传 iPhone/iPad 拍摄的 `.heic` / `.heif` 文件
2. 选择输出格式：**JPG** 或 **PNG**
3. 点击 **「转换并下载」**

> ⚠️ 首次使用需加载解码引擎（约 1–2 秒），后续使用即时可用。

## 浏览器兼容性

| 浏览器 | 最低版本 | JPG/PNG/WebP | AVIF | HEIC |
|--------|----------|:---:|:---:|:---:|
| Chrome | 85+ | ✅ | ✅ | ✅ |
| Firefox | 93+ | ✅ | ✅ | ✅ |
| Safari | 14+ | ✅ | ❌ | ✅ |
| Edge | 85+ | ✅ | ✅ | ✅ |

- **AVIF** 在 Safari 上不可用，按钮会自动变灰并提示
- **HEIC** 在 iOS Safari 上可能直接由系统解码，无需加载 WASM

## 隐私与安全

- 🔒 **零上传**：图片仅在本地浏览器处理，不经过任何服务器
- 💾 **零存储**：不保存任何图片数据到服务器或本地存储
- 🍪 **Cookie**：仅用于广告展示和流量统计，不收集个人信息
- 🔌 **断网可用**：首次加载后，即使断开网络也能正常使用

## 常见问题

### 转换后的图片在哪里？
浏览器自动下载到默认下载文件夹。若未自动下载，请检查浏览器是否拦截了弹窗。

### 为什么 AVIF 按钮是灰色的？
当前浏览器不支持 AVIF 格式。请使用 Chrome 85+ 或 Firefox 93+。

### 转换后图片比原图大？
这通常发生在将低质量 JPG 转为 PNG 时，PNG 是无损格式所以体积更大。可以降低质量滑块数值，或选择 WebP 格式来减小体积。

### 可以转换其他格式吗（如 GIF/TIFF/BMP）？
当前版本支持 JPG/PNG/WebP/AVIF/ICO 输出。如需 GIF 等更多格式，请关注后续更新。

### 批量转换最多支持多少张？
一次最多 10 张。如果超过 10 张，请分批转换。

### HEIC 转换为什么比其他格式慢？
HEIC 解码依赖 WASM 引擎，首次使用需加载约 1.2MB 的解码库。加载完成后，单个 HEIC 文件转换通常在 3 秒以内。

## 部署说明

本工具为纯静态单 HTML 文件，可部署到任何静态托管服务：

- **Cloudflare Pages**：推送至 GitHub → Cloudflare Dashboard 一键部署
- **GitHub Pages**：上传至仓库 → Settings → Pages 启用
- **Vercel / Netlify**：直接拖拽 `index.html` 即可部署
- **本地使用**：双击 `index.html` 直接用浏览器打开

### 配置项（部署后）

1. **Cloudflare Web Analytics**：替换 HTML 末尾的 `YOUR_TOKEN_HERE`
2. **Google AdSense**：替换 4 个广告占位 `div` 为 AdSense 代码
3. **域名/Canonical URL**：修改 `<link rel="canonical">` 和结构化数据中的 URL

## 技术架构

```
用户上传 → FileReader → Image 对象 → Canvas 渲染 → toBlob() → 下载
                                                    ↕
                                              WASM（HEIC 解码 / AVIF 编码）
```

- **核心引擎**：浏览器 Canvas API
- **HEIC 解码**：heic2any (libheif WASM)，按需加载
- **ICO 编码**：自实现多尺寸 PNG → ICO 容器封装
- **ZIP 打包**：JSZip (CDN)
- **图标库**：Iconify (Lucide 图标集)
